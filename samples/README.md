# Sample Data

This folder contains fictional documents used to test Coverage Clarity.

- `contract-required-full.txt` - contract with multiple insurance requirements
- `contract-no-wos.txt` - contract without a waiver-of-subrogation requirement
- `coi-present-strong.txt` - certificate with several matching evidence signals
- `coi-unmet-limits.txt` - certificate with limits below the contract requirement
- `coi-missing-wos.txt` - certificate without waiver-of-subrogation evidence
- `policy-present-strong.txt` - policy-style supporting evidence
- `contract-management-lines.txt` - management liability requirements
- `policy-management-lines-partial.txt` - partial D&O/EPLI/crime evidence with missing or inadequate management lines
- `intake-request.json` - structured analysis request example

All sample data is fictional and intended for prototype testing only.
